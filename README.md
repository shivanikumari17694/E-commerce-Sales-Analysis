# E-commerce-Sales-Analysis

# 📊 E-commerce Sales Analysis using SQL

## 📌 Project Overview

This project focuses on analysing e-commerce sales data using SQL.
I designed a relational database with Customers, Products, and Orders tables and performed data analysis to generate meaningful business insights.

---
## 🛠️ Tools & Technologies

* MySQL Workbench
* SQL
---

## 📂 Database Structure
### 1. Customers Table

* customer_id (Primary Key)
* name
* city

### 2. Products Table

* product_id (Primary Key)
* product_name
* category
* price

### 3. Orders Table

* order_id (Primary Key)
* customer_id (Foreign Key)
* product_id (Foreign Key)
* quantity
* order_date
---

## 🔗 Relationships

* Customers ↔ Orders (One-to-Many)
* Products ↔ Orders (One-to-Many)

---

## 📊 Key Analysis Performed

* ✔ Total Revenue Calculation
* ✔ Top 5 Customers by Spending
* ✔ Best-Selling Products
* ✔ Sales by Category
* ✔ City-wise Revenue Analysis
* ✔ Monthly Sales Trends
